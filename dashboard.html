RESULT
달릴 때 막아주는 판 (진흙 방지판)
달리는 말 발굽의 진흙·돌을 막아주는 마차 앞 판자. 탑승자 보호 목적으로 '차단'이 핵심 기능. 단어 어원상 가장 원초적 의미.
f₂ · 유추이미지
자동차 문화 · 20세기
Automobile
f₂(자동차문화) × X(직관파악판) →
RESULT
계기판 (운전석 상태 표시판)
마차 판의 위치 그대로 자동차 운전석에 이식. '차단'이 '상태 표시'로 전환됨. 달리는 차 앞 → 달리는 차 상태 표시판.
f₃ · 확장이미지
IT·경영 문화 · 21세기
IT / Business
f₃(IT·경영문화) × X(직관파악판) →
RESULT
상황판 (통합 데이터 모니터링 화면)
DBMS·ERP·재무 등 전사 데이터를 한눈에 파악하는 화면. 움직이는 프로젝트·기업 앞 전체 상황판. '계기판 보며 방향 잡기' 비유 그대로 계승.
COMMON PATTERN — 공통 핵심 이미지
운동하는 시스템(말·자동차·기업) 앞에 위치하여
그 현재 상태를 직관적으로 보여주는 표면
마차 → 자동차 → IT·경영, 세 시대 모두 '무언가가 달리거나 작동하는 것 앞의 판'이라는 이미지를 공유한다.
한국어로 풀면 달릴 때 막아주는 판 → 계기판 → 상황판 으로 변천하지만,
맥락 f가 바뀌어도 핵심 이미지 X(판)는 그대로 유지된다 — 이것이 F(X)Voca의 핵심 원리.
[ 3 ] 구별 개념 — 혼동하기 쉬운 유사 용어
dashboard vs.
Report
보고서
보고서는 과거 데이터를 정리한 정적(靜) 문서. 대시보드는 현재 상태를 실시간 시각화하는 동적(動) 화면.
→ 핵심 차이: 시제(時制)
dashboard vs.
Monitor
모니터·관제 화면
모니터는 단순 데이터 표시. 대시보드는 의사결정용으로 가공·요약, KPI 중심 배치.
→ 핵심 차이: 가공·요약 여부
dashboard vs.
EIS
경영정보시스템
EIS는 데이터 수집·처리·저장·표시 시스템 전체. 대시보드는 EIS의 최종 출력 화면 레이어.
→ 핵심 차이: 범위(시스템 전체 vs. 화면)
dashboard vs.
BI Tool
BI 솔루션
BI(Business Intelligence)는 데이터 분석 기술·도구 전반. 대시보드는 BI 결과물을 보여주는 시각화 인터페이스.
→ 핵심 차이: 도구 vs. 결과물 화면
[ 4 ] 현재 활용 유형 — 확장이미지(상황판) 하위 분류
01 /
BI 대시보드
Business Intelligence Dashboard
ERP·DBMS 데이터를 분석하여 경영진 의사결정을 지원. Tableau, Power BI 등이 대표 도구.
02 /
통합 모니터링 시스템
Integrated Monitoring System
서버·네트워크·DB 상태를 IT 인프라 관점에서 한 화면에 통합 표시.
03 /
경영정보시스템
EIS · Executive Information System
임원·경영자가 매출·재고·자금 주요 지표를 실시간 파악하는 전통적 시스템.
04 /
관제 센터 / 워룸
War Room · Control Center
긴급 상황 대응 목적의 실시간 전사 현황 화면. 보안·제조 분야에서 주로 사용.
